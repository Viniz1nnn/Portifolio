Um pouco sobre mim

Olá! Sou um entusiasta da tecnologia e um desenvolvedor front-end de 18 anos. Minha jornada na programação começou no início deste ano, e, desde então, tenho mergulhado de cabeça no aprendizado de HTML, CSS e JavaScript.

Desde muito jovem, sempre tive um forte desejo de trabalhar na área de tecnologia, e ao descobrir a programação, encontrei a forma perfeita de expressar minha criatividade e resolver problemas de maneira inovadora.

Estou em busca de desafios que me permitam aplicar meus conhecimentos e continuar evoluindo como desenvolvedor. Meu objetivo é criar experiências web impactantes e envolventes que façam a diferença.

StarBucks

Este projeto é uma réplica detalhada do site da Starbucks, desenvolvida como um exercício prático para fortalecer o conhecimento em desenvolvimento web.

Utilizamos tecnologias como HTML e CSS para reproduzir a interface e funcionalidades do site original. Este trabalho foi realizado sob a orientação do Professor Bonieky Lacerda, no curso da B7Web.

Pizzas

Este projeto é um site de pizzaria com funcionalidades de seleção e adição de pizzas ao carrinho, desenvolvido com JavaScript, HTML e CSS. Foi realizado como parte do curso de ensino e aprendizagem com o Professor Bonieky Lacerda da B7Web.

O objetivo principal foi aplicar na prática conceitos aprendidos, proporcionando uma interface interativa e amigável para o usuário. Através deste projeto, aprimoro minhas habilidades, técnicas e a capacidade de criar experiências digitais envolventes e funcionais.
